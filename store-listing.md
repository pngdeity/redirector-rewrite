## Chrome Web Store

### Short Description (132 chars max)
"Redirect URLs based on user-defined wildcard or regex patterns. A clean, lightweight Manifest V3 rewrite of the classic Redirector extension."

### Detailed Description
Redirector Rewrite intercepts outbound web requests and seamlessly redirects them based on your custom rules. Define patterns using wildcards or regular expressions to automatically reroute URLs to your preferred destinations.

**Features:**
- Wildcard (`*`) and Regular Expression pattern matching with capture group substitution (`$1`, `$2`, etc.)
- Include/exclude patterns for fine-grained control
- Match processing: URL encoding/decoding, Base64 decoding
- Live rule testing with infinite-loop detection as you type
- Storage sync for cross-device rule sharing
- Organize mode with visual grouping and batch operations
- Enable/disable per-rule or globally
- Import/export rules as JSON
- Adaptive dark mode icons and badge indicators
- SPA navigation interception (HistoryState)

**Privacy:** All rules and settings are stored locally in your browser. No data is collected or transmitted.
